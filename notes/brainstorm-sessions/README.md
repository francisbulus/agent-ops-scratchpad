# Brainstorm Sessions

One markdown file per session/day.

## Naming

- Format: `NNN-YYYY-MM-DD.md`
- Example: `001-2026-02-07.md`

## How to use

- Create one new file each day/session.
- Increment `NNN` by 1.
- Keep decisions and open questions short and explicit.

## Session Template

```md
# Session NNN - YYYY-MM-DD

## Goal

-

## MVP Scope

-

## Must-Have Features

-

## Not in MVP

-

## Risks

-

## Open Questions

-

## Next Actions

-
```
